Welcome Bitcoin Classroom - Western Mass

Learn to use Bitcoin Hardware in a Classroom Environment with an Experienced Instructor

There is an inordinate amount of online learning material available to teach bitcoiners to be more self-sovereign, however not everyone learns best from online material alone. Bitcoin Classroom intends to offer classes for students who want to ask questions about why and how things work when interacting with hardware and who want help with troubleshooting when things don’t go the way they watch online. 

The focus of Bitcoin Classroom is not to teach the “why” of bitcoin, but to teach the “how” of bitcoin. Specifically, Bitcoin Classroom classes are focused on teaching students how to use bitcoin in sovereign ways. This means withdrawing bitcoin off of exchanges and into wallets where the keys are held offline (i.e., hardware wallets) in single or multi-signature configurations using open-source software. 

Signing up for a class reserves you a seat in that class (limited to 5 students per class). Each student can choose from any of the following topics to learn during the class and each student can choose a different topic:

  • Multisig-101: Learn to setup a multisig wallet in Sparrow using a ColdCard, Ledger Nano S and Trezor One
  • Multisig-201: Learn to restore a multisig wallet in Sparrow and Nunchuk in a variety of ways and learn to encrypt config files.
  • Single Sig Hardware Wallet Tutorials: Ledger Nano S, ColdCard Mk4, Trezor One, Blockstream Jade
  • Setting up a Bitcoin Full Node (Start9): Setup a Start9 full node using a repurposed HP mini PC. Option to purchase node hardware to setup and take home with you.

Laptops, hardware wallets and/or node hardware will be provided for in class use. In some cases, hardware can be purchased separately. The instructor will reach out to you prior to the class to see which topics you want to cover in the class.
